# Gift Matrix — Interactive Web Brochure

A premium, minimal, and elegant interactive web brochure for **Gift Matrix**, a luxury gifting brand that transforms gifting into meaningful, story-led experiences.

---

## 🎯 Project Goal
Create a polished, luxury brochure-style website that:
- Showcases the brand identity of Gift Matrix
- Presents all brand content in a structured, flowing sequence
- Displays project photos in a continuously moving horizontal reel animation
- Feels premium, refined, and emotionally resonant

---

## 📁 File Structure
```
index.html          — Main brochure (all sections)
css/
  style.css         — Full luxury styling (Ivory, Gold, Black palette)
js/
  main.js           — Animations, FAQ accordion, reel controls, effects
README.md           — This file
```

---

## ✅ Completed Features

### Pages (in strict order)
1. **Page 1 — Introduction**: Hero section with brand tagline, philosophy, CTA
2. **Page 2 — Core Belief**: 3-card belief system with elegant layout
3. **Page 3 — Our Core Services**: 4 service cards + **Horizontal Photo Reel**
4. **Page 4 — Why Gift Matrix**: Dilemma infographic, Who We Serve grid (6 personas), Occasions Wheel, 4-step gifting journey process flow
5. **Page 5 — Why Us**: 3-column approach grid, brand philosophy statement, testimonials placeholder
6. **Page 6 — Call to Action**: Full-screen CTA with contact details and gold particle effects
7. **FAQ Section**: Animated accordion with 6 questions
8. **Comparative Analysis**: Side-by-side table (Traditional vs Gift Matrix)

### Design & Interactions
- ✅ Sticky navigation with scroll fade-in effect
- ✅ Mobile hamburger menu
- ✅ Scroll reveal animations (staggered)
- ✅ Horizontal auto-scrolling photo reel with pause/play controls
- ✅ Draggable reel (mouse + touch)
- ✅ Occasions Wheel visual (8 occasions radially arranged)
- ✅ FAQ accordion (open/close with smooth animation)
- ✅ Floating gold particle effect on CTA page
- ✅ Parallax fade effect on hero scroll
- ✅ Active nav link highlighting

### Design System
- **Colors**: Ivory `#F8F4ED`, Gold `#C9A84C`, Black `#0A0A0A`, Beige `#E8DFD0`
- **Fonts**: Cormorant Garamond (serif, headings) + Jost (sans-serif, body)
- **Style**: Minimal, luxury, premium — no clutter
